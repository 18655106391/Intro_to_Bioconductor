Package ShortRead deals with fasta and fastq file.

fasta file

Each sequence in fasta format takes 2 lines:
> unique_sequence_identifier
raw_sequence_string

library(ShortRead)

# read fasta
fasasample <- readFasta("covid19_sequence.fasta")
fasasample
## class: ShortRead
## length: 1 reads; width: 29800 cycles
id(fasasample)
##   A BStringSet instance of length 1
##     width seq
## [1]   117 MT676397.1 Severe acute respirator...IND/GBRC189/2020, complete genome
length(fasasample)
## [1] 1
width(fasasample)
## [1] 29800

So we read in one read with length 29800, named 117MT676397.

fastq

@ unique_sequence_identifier
raw_sequence_string
+ optional id
quality encoding per sequence letter

Sequence quality

Filter

Filters can be used to subset a ShortReadQ object, or directly to a fastq file.

nFilter() has a threshold parameter, representing the maximum number of Ns allowed on each read.

my_filter <- nFilter(threshold=10)
fq_sample[my_filter(fq_sample)]
## class: ShortReadQ
## length: 5 reads; width: 50 cycles
# use the filter when reading
filtered_fq <- readFastq("SRR1971253.fastq",filter=my_filter)
filtered_fq
## class: ShortReadQ
## length: 500 reads; width: 50 cycles

idFilter() uses a regular expression to select specific ids.

my_idfilter <- idFilter(regex=":3:1")
filtered_fq <-readFastq("SRR1971253.fastq",filter=my_idfilter)
filtered_fq
## class: ShortReadQ
## length: 257 reads; width: 50 cycles
ShortRead::id(filtered_fq)[1]
##   A BStringSet instance of length 1
##     width seq
## [1]    62 SRR1971253.15442 HS2:546:C4DUWACXX:3:1210:6648:68689 length=50

polynFilter() accepts a maximum of certain number of continuous letters per sequence. For example, we can create a filter to remove sequences that has more than 10 continuous As.

my_polyA_filter <- polynFilter(threshold=10,nuc=c("A"))
filtered_fq <-readFastq("SRR1971253.fastq",filter=my_polyA_filter)
filtered_fq
## class: ShortReadQ
## length: 198 reads; width: 50 cycles

srFilter() is a function to create our own filter.

read_with_cutoff <- srFilter(function(x) {width(x)>minWidth},name='MinWidth')
minWidth <- 51
fq_sample[read_with_cutoff(fq_sample)]
## class: ShortReadQ
## length: 0 reads; width:  cycles
